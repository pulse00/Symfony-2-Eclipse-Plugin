The plugin gives its best to determine when to show the correct contextual code-assist proposals. Wherever it can't do this, you can manually insert Symfony specific values in your code using dialogs. All of the dialogs are also available via keyboard shortcuts - the keybindings are configurable via the regular Eclipse keyboard shortcut dialog.

Insert Route

Press Ctrl+K to open the route insertion dialog (also available via the main menu Source -> Insert Route)

Insert Service ID

Press Ctrl+S to open the service insertion dialog (also available via the main menu Source -> Insert Service ID)

Note that these dialogs are available in any editor - e.g. for inserting service IDs inside your XML configuration.
