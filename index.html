Symfony-2-Eclipse-Plugin by pulse00

An extension to the PDT Plugin of eclipse to support Symfony 2 development

The plugin is - like Symfony 2 itself - still under development. We target a release date together with the final release of Symfony 2. Please feel free to add feature wished to the feature-list in the wiki: http://wiki.github.com/pulse00/Symfony-2-Eclipse-Plugin/feature-list
